# 기사콜

배달·물류 기사용 초간단 모바일 웹앱입니다. "기사매니저"와 비슷한 결의 초경량 도구예요.

## 뭐 하는 앱인가요?
1. 앱을 열면 **남자 / 여자** 성별 선택 화면이 뜹니다.
2. 성별을 고르면 다음 화면으로 넘어가고, 큼직한 버튼 하나가 보입니다.
   - **"기사님 오늘 물량 15카트넘어요!"**
3. 그 버튼을 누르면 사이렌 알림 연출이 크게 뜹니다. 그게 전부예요.

## 특징
- **상태 저장 없음**: localStorage 등 누적 저장 기능을 일부러 넣지 않았습니다. 새로고침하면 처음부터.
- **다크 마스큘린 테마**: 차콜 배경 + 골드/딥레드 포인트, 미세 노이즈 텍스처.
- **모바일 반응형**: 폰 화면에 꽉 차게, 큰 터치 버튼.
- **단일 파일 자족형**: `index.html` 하나에 HTML/CSS/JS 인라인. 외부 의존성 0.

## 실행
- 로컬: `index.html`을 브라우저로 열면 끝.
- 배포: GitHub Pages (main 브랜치 root).

## 앱처럼 쓰기 (PWA)
- 폰 브라우저에서 라이브 URL 접속 → 메뉴의 **"홈 화면에 추가"** 선택.
- 그러면 주소창·링크바 없이 **전체화면 일반 앱처럼** 실행됩니다. (일반 브라우저 탭으로 열면 주소창은 원래 안 사라지는 게 정상 — 홈 화면 아이콘으로 실행해야 전체화면)
- 설치형 요건(서비스워커 + manifest `display:standalone`)을 충족하므로 홈 화면 추가/설치가 가능합니다.
- 홈 화면 아이콘: 다크 차콜 배경 + 골드 트럭 + 딥레드 콜(전화) 배지.
- 아이콘이 안 보이면 브라우저 캐시 때문일 수 있어요. 서비스워커 캐시 버전(gisacall-v5)과 순수 파일명(쿼리스트링 없음)으로 최신 아이콘을 받도록 처리했습니다.

## 실제 푸시 알림 (사장님용 · ntfy.sh)
기사가 **"기사님 오늘 15카트넘어요!"** 버튼을 누르면, 사장님 폰으로 **앱을 안 켜도** 푸시 알림이 옵니다.
백엔드 없이 [ntfy.sh](https://ntfy.sh) 무료 서비스로 동작합니다.

### 사장님 폰 설정 (딱 한 번, 3단계)
1. 폰에 **ntfy** 앱 설치 (Play스토어 / App Store에서 "ntfy" 검색)
2. 앱에서 **토픽 구독** 추가 → 토픽 이름에 정확히 입력:
   `gisacall-donglee-kd15cart-7f3k9x`
3. 끝! 이제 기사님이 버튼을 누르면 사장님 폰에 알림이 뜹니다.

- 기사님(어르신)은 **아무 설정도 필요 없습니다.** 그냥 버튼만 누르면 됩니다.
- 토픽 문자열은 추측하기 어렵게 만든 고정값이라, 아는 사람만 알림을 받습니다. (완전 비공개는 아니니 민감정보는 넣지 않음)
- 앱 화면 하단의 접이식 **"⚙ 사장님 설정 안내"**에도 같은 3단계가 들어 있습니다.

### 기술 메모
- 전송: 버튼 클릭 시 `POST https://ntfy.sh/<토픽>` (Title/Priority/Tags 헤더 + 한글 본문).
- fetch는 `try/catch`로 감싸 실패(오프라인 등)해도 화면 흐름(알림+감사멘트)은 정상 진행됩니다.
- 헤더 값은 ASCII만 허용되므로(브라우저 제약) 제목은 `GisaCall`, 한글은 본문(UTF-8)에 담습니다.
