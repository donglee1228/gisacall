# 기사콜

배달·물류 기사용 초간단 모바일 웹앱입니다. "기사매니저"와 비슷한 결의 초경량 도구예요.

## 뭐 하는 앱인가요?
1. 앱을 열면 **남자 / 여자** 성별 선택 화면이 뜹니다.
2. 성별을 고르면 다음 화면으로 넘어가고, 큼직한 버튼 하나가 보입니다.
   - **"기사님 오늘 물량 15카트넘어요!"**
3. 그 버튼을 누르면 **오늘 근무 중인 기사에게 바로 전화(tel:)가 걸립니다.** (사이렌 알림 연출도 함께 표시)

## 격일제 근무 · 자동 전환
- 두 기사가 하루씩 번갈아 근무합니다.
  - 2026-07-05 = **김종대 기사** 근무 / 이기동 기사 쉼
  - 2026-07-06 = **이기동 기사** 근무 / 김종대 기사 쉼
  - 이후 하루 단위로 무한 교대 (기기 로컬 날짜 자정 기준 전환)
- "15카트넘어요!" 버튼 → 그날 근무자에게 전화 (김종대 근무일=010-3383-5808 / 이기동 근무일=010-3805-9275).
- 하단 두 전화 카드 번호 밑에 오늘 상태가 자동 표시됩니다: 근무자 **(일하는중)** 초록, 쉬는 기사 **(안하는중)** 회색.
  앱을 계속 켜둬도 60초마다·다시 포커스 시 재계산해 자정이 지나면 자동으로 바뀝니다.
- **근무일은 항상 한국시간(KST) 기준**으로 계산합니다. 해외/시간대가 잘못 설정된 폰에서도 한국 날짜로 판정합니다.

### 특정 날 근무자 바꾸기 (예외표)
- 누가 갑자기 쉬는 날엔 `index.html` 상단의 `예외근무` 맵에 한 줄만 추가하면 됩니다:
  ```js
  var 예외근무 = {
    '2026-07-10': 'kim',   // 그날은 김종대(kim) 근무 / 이기동은 'lee'
  };
  ```
- 날짜는 `'YYYY-MM-DD'`(한국날짜), 값은 `'kim'`(김종대) 또는 `'lee'`(이기동).
- 예외표가 격일제 계산보다 **우선**하며, 15카트 전화 대상과 (일하는중)/(안하는중) 표시에 **동시 반영**됩니다. 고친 뒤 재배포하면 모든 폰에 동일 적용.

### 사장님용 근무표 수정 패널 (앱 안에서, AI 없이)
- 전화 카드 아래 **"🔒 근무표 수정 (사장님)"** 버튼 → 비밀번호 `001234` 입력 → 수정 화면.
- 날짜(`YYYY-MM-DD`) 선택 + 근무자(김종대/이기동) 고르고 **[이 날짜 저장]** → 그날은 격일제와 상관없이 지정한 사람이 근무자.
- 저장된 예외 목록에서 **[삭제]** 하면 그날은 다시 기본 격일제로 복귀.
- **⚠ 이 저장은 "이 폰에서만" 적용됩니다** (브라우저 localStorage, 키 `gisacall_예외근무`). 기사님들 폰에는 반영 안 됩니다.
  기사님 폰들은 기본 격일제(+코드 하드코딩 `예외근무` 맵)로 동작합니다.
- **전 폰 공통으로 바꾸려면** 위의 코드 `예외근무` 맵을 고쳐 재배포해야 합니다.
- 근무자 판정 우선순위: ① 이 폰 예외(패널 저장) → ② 코드 하드코딩 예외 → ③ 기본 격일제(KST).

## 특징
- **상태 저장 없음**: localStorage 등 누적 저장 기능을 일부러 넣지 않았습니다. 새로고침하면 처음부터.
- **다크 마스큘린 테마**: 차콜 배경 + 골드/딥레드 포인트, 미세 노이즈 텍스처.
- **모바일 반응형**: 폰 화면에 꽉 차게, 큰 터치 버튼.
- **단일 파일 자족형**: `index.html` 하나에 HTML/CSS/JS 인라인. 외부 의존성 0.

## 실행
- 로컬: `index.html`을 브라우저로 열면 끝.
- 배포: GitHub Pages (main 브랜치 root).

## 앱처럼 쓰기 (PWA)
- 폰 브라우저에서 라이브 URL 접속 → 메뉴의 **"홈 화면에 추가"** 선택.
- 그러면 주소창·링크바 없이 **전체화면 일반 앱처럼** 실행됩니다. (일반 브라우저 탭으로 열면 주소창은 원래 안 사라지는 게 정상 — 홈 화면 아이콘으로 실행해야 전체화면)
- 설치형 요건(서비스워커 + manifest `display:standalone`)을 충족하므로 홈 화면 추가/설치가 가능합니다.
- 홈 화면 아이콘: 다크 차콜 배경 + 골드 트럭 + 딥레드 콜(전화) 배지.
- 아이콘이 안 보이면 브라우저 캐시 때문일 수 있어요. 서비스워커 캐시 버전(gisacall-v6)과 순수 파일명(쿼리스트링 없음)으로 최신 아이콘을 받도록 처리했습니다.
